# Sample Report

This directory will contain the professional example deliverable produced after the vulnerable demo application has been audited and repaired.

Expected sections:

- executive summary
- scope and exclusions
- launch verdict
- severity-ranked findings
- technical evidence
- reproduction steps
- business impact
- recommended remediation
- verification status
- prioritized action plan

Every finding must be reproducible or supported by concrete evidence. Avoid claims of full penetration testing, compliance, or guaranteed vulnerability detection.
