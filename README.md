# AI Production Readiness

A 12-month project to help founders and small teams turn AI-built web applications into software that is safer, more reliable, testable, and ready for real users.

## Current stage

Stage 0 — capability, proof, and first paid review.

## Immediate objective

Complete one legitimate paid AI-Built App Launch Readiness Review.

## Source of truth

Read [`PROJECT_STATE.md`](./PROJECT_STATE.md) before making major project decisions or starting substantial work.

## Initial technical focus

- Next.js / React / TypeScript
- Supabase or Firebase
- Vercel
- Stripe
- OpenAI or Anthropic APIs
- GitHub

## Repository structure

- `demo-app/` — deliberately vulnerable demonstration application
- `audit-checklist/` — repeatable review methodology and verification criteria
- `sample-report/` — example client-facing findings and remediation report
- `research/` — evidence that directly supports technical delivery or customer validation

## Security rule

Do not commit API keys, credentials, real customer data, client repositories, or other sensitive material. The repository must be private before storing client-sensitive project artifacts.
